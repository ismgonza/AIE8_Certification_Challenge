# ISO RAG Frontend

Simple React + Vite + Tailwind frontend for the ISO RAG system.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:5173`

**Make sure the backend is running on `http://localhost:8000`**
